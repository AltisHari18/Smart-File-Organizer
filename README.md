# Smart File Organizer

## Project Overview
Smart File Organizer is a Python automation project that helps users organize files automatically into different folders based on their file extensions.

The program scans a selected directory, identifies file types, creates folders if needed, and moves files into their respective categories such as Images, Videos, Documents, Music, and Others.

---

## Features
- Automatic file organization
- Folder creation
- File type detection
- Categorizes files automatically
- Simple and easy to use
- Reduces folder clutter

---

## Technologies Used
- Python
- OS Module
- Shutil Module

---

## File Categories
The program organizes files into:
- Images
- Videos
- Documents
- Music
- Programs
- Archives
- Others

---

## How It Works
1. User enters the folder path
2. Program scans all files
3. File extensions are detected
4. Required folders are created automatically
5. Files are moved into respective folders

---

## Example
Before Organization:
Downloads/
- photo.jpg
- movie.mp4
- notes.pdf

After Organization:
Downloads/
- Images/photo.jpg
- Videos/movie.mp4
- Documents/notes.pdf
